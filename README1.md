- 单引号和双引号的区别
    - 单引号不解析变量；单引号只解析\和\\
    - 双引号解析变量；双引号解析所有的转义字符      

- {$var}和${var}都可以用来替换变量，建议用第一个

- heredoc和nowdoc
    - heredoc相当于双引号的作用,语法如下
    ```
    $heredoc = <<<EOF(名字可随意选取，但要配对，并且结尾不能有空格等多余的内容,可以用双引号包裹，不能用单引号哦)
        需要填充的内容
    EOF;
    ```
    - nowdoc相当于单引号的作用，语法如下
    ```
    $nowdoc = <<<'EOD'(必须用单引号包裹)
        需要填充的内容
    EOD;
    ```
